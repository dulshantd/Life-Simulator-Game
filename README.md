## 🌟 About the Project

Welcome to the **Life Simulation Game**, a terminal-based Java project where every decision shapes your character’s destiny.

You start as a young individual and progress through life by making choices in:
- 💼 Career
- 🎓 Education
- 💰 Finance
- ❤️ Health
- 😊 Happiness
- 🎲 Risk & Gambling

Every action has consequences — just like real life!

---

## 🚀 Features

✨ Fully interactive console-based gameplay  
💰 Money & economy system  
❤️ Health & happiness tracking  
🧠 Intelligence system  
🎲 Gambling & risk management system  
📈 Investment & stock simulation  
🎯 Random life events system  
🏆 Win / loss conditions  
🎨 Colorful terminal UI using ANSI codes
